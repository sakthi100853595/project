This project is a full-featured e-commerce web application built with a modern JavaScript stack: React.js and Next.js for the frontend, Node.js and Express.js for the backend, and MongoDB as the database. 
Features:
> User registration, login, and authorization with secure authentication
> Product catalog with CRUD operations from the admin dashboard.
> Shopping cart: Add, remove, and update items, with dynamic pricing.
> Product search, filtering, reviews, and ratings
> Admin dashboard for managing products, users, and orders.
