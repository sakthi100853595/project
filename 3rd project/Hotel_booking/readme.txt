This project is a responsive and user-friendly hotel booking frontend application built with React.js and styled using CSS. It enables users to browse available hotels, view detailed information about rooms, select dates, and make bookings seamlessly through an intuitive interface
Key Features:
> Display hotel listings with key information including name, location, rating, and price
> Hotel detail pages with room descriptions, images, and booking forms
> Date selection and booking form to reserve rooms
> Form validation to enhance user input and booking accuracy
